Introduction

The following links are provided for anyone who wants to access to the data used in the paper "Detecting Iterative Publications of Phishing Sites". All the files are in the JSON format, except the DOM folders, which contain three types of files:
1) screentshot files (*.png),
2) URL files (*.url),
3) DOM files (*.html).
The URL files (*.url) include two lines: the first line is the first URL of the page, the second line is the final URL of page. In the paper, all the URLs mentioned in the "Experiment" section are the final URLs. The prefix of file names in the DOM folder is the MD5 of the first URL. To facilitate the work of dealing with data, except the links in "URLs" section, all the other links provided in this page are the list of files in corresponding DOM folder.
