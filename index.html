Introduction

The following links are provided for anyone who wants to follow our ideas in paper "Detecting Iterative Publications of Phishing Sites", all the files are in JSON format, except the DOM folder, which is organized with three types of files: 1) screentshot file(*.png), 2)URL file(*.url), 3)DOM file(*.html). URL file(*.url) includes two lines. The content in the first line is the first URL of page, and the second line is the final URL of page. In the paper, all the URLs mentioned is the final URL. The prefix of file name in DOM folder is the MD5 generated by the first URL of page. To facilitate the work of dealing with data, except files in URL section, all the other links provided are the list of files in corresponding DOM folder instead of URL.
